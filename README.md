# Keys For Sketch

<br />
Keyblade is a full-featured shortcut manager for Sketch. Integrated directly to Preferences window it helps you to customize shortcuts easily.
<br />
<br />

## Installation
Download latest release of Keyblade and double-click it.

## Planned Features
* Easily menu shortcuts customization
* Single-character shortcuts customization (like Pencil or Vector) 
* Shortcut conflicts resolving
* Any third-party plugins support
* Preferences window integration
* Intuitive lightning-fast UI
* Instant menu search
